## 🚀 TK-SX چیست؟

**TK-SX** یک **پنل مدیریت VPN نسل جدید** است که با هسته‌ی قدرتمند **Sing-box** کار می‌کند. این پنل یک داشبورد وب حرفه‌ای، یک **ربات فروش کامل تلگرام (Cyrus Bot)** با تمام قابلیت‌های میرزا، و یک **مینی‌اپ زیبا** برای مدیریت موبایل را در یک بسته‌ی یکپارچه گرد هم آورده است.

طراحی شده برای **Railway** و قابلیت مدیریت **۱۵ کاربر هم‌زمان** با مصرف منابع بسیار کم.

---

## ✨ ویژگی‌ها

### 🔥 هسته اصلی پنل
- **۱۱ پروتکل VPN** — VLESS, VMess, Trojan, Shadowsocks, SOCKS5, HTTP, WireGuard, Hysteria2, TUN, Dokodemo-door, Snell
- **UDP over TCP** برای WireGuard و Hysteria2 (دور زدن محدودیت UDP ریلی)
- **XHTTP Ultra** با دو حالت packet-up و stream-up
- **مانیتورینگ اتصالات زنده** با نمایش IP
- **گروه‌های ساب‌اسکریپشن** با صفحه پابلیک و رمز
- **نمودارهای ترافیک** — ساعتی، روزانه، پیک مصرف
- **رابط کاربری فارسی/انگلیسی** با تم تاریک/روشن

### 💰 کیف پول و پرداخت‌ها
- **سیستم کیف پول داخلی** با تاریخچه تراکنش
- **پرداخت کارت به کارت** با تأیید رسید
- **درگاه ریمیت** (قابل تنظیم)
- **پشتیبانی از استارز تلگرام** (نرخ قابل تنظیم)
- **پرداخت با گیفت** (ارسال به @ItzJustEren)

### 🔗 سیستم معرفی و بازاریابی
- **سیستم معرفی** با ۱۰٪ کمیسیون
- **کد معرفی اختصاصی** برای هر کاربر
- **کد تخفیف** (ساخت توسط ادمین با محدودیت استفاده)
- **سیستم قرعه‌کشی** با بلیت به ازای هر خرید

### 🤖 ربات Cyrus Bot
- **خرید خودکار** محصولات
- **تست رایگان** (۵۰ مگابایت، هفته‌ای یک بار)
- **تمدید اشتراک** با یک کلیک
- **سطح‌بندی کاربران** (سطح ۱۰ = ۵ گیگ رایگان)
- **تأیید/رد سفارش** با دلیل (رسید فیک، دیرتر از ساعت)
- **بازخورد کاربران** با تأیید ادمین
- **پنل ادمین کامل** در تلگرام

### 🖥️ مینی‌اپ (داشبورد موبایل)
- **صفحه ورود** با پیام خوش‌آمدگویی
- **آمار لحظه‌ای** — اتصالات، کانفیگ‌ها، محصولات، سفارشات
- **دسترسی سریع** به پنل اصلی و ربات
- **تم آبی-یخی-مشکی** با لوگوی عقاب
- **کاملاً واکنش‌گرا** برای موبایل

## ⚙️ متغیرهای محیطی

| متغیر | توضیح | پیش‌فرض |
|-------|-------|---------|
| `PORT` | پورت وب | `8000` |
| `TELEGRAM_BOT_TOKEN` | توکن ربات از @BotFather | — |
| `ADMIN_PASSWORD` | رمز عبور پنل ادمین | `taakaa` |
| `CARD_NUMBER` | شماره کارت پیش‌فرض | `6037-9910-1234-5678` |
| `CARD_OWNER_NAME` | نام صاحب کارت | `نام صاحب کارت` |
| `PRICE_PER_GB` | قیمت هر گیگ (به هزار تومان) | `6` |
| `REQUIRED_CHANNEL` | کانال الزامی برای کاربران | `@TaaKaaOrg` |
| `ADMIN_GROUP_ID` | آیدی گروه برای ثبت کارت‌ها | — |
| `STARS_RATE` | نرخ استارز تلگرام (تومان/استارز) | `1000` |
| `BOT_USERNAME` | نام کاربری ربات برای لینک‌های معرفی | `CyrusBot` |
